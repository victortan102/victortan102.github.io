Project Description

The Smart Aquaponics System project is an innovative IoT-based solution designed to integrate aquaculture (fish farming) with hydroponics (growing plants in nutrient solutions). With urban agriculture trends on the rise, the system addresses the critical need for stable water temperature control to maintain fish health. The solution enables farmers to monitor and manage farm conditions remotely using smart devices, ensuring a consistent and conducive environment for both fish and plants.

System Operation (How the program works)

1. Core Features

  • Temperature Monitoring: Real-time water temperature data is captured using a BME280 sensor connected to a Raspberry Pi.
  • Alerts and Notifications: The system sends alerts when the water temperature exceeds a user-defined threshold.
  • Remote Control: Farmers can turn on or off a cooling fan via an MQTT-based mobile application to maintain optimal temperature conditions.

2. System Components

  • Hardware: Raspberry Pi 3 Model B acts as the edge device, interfaced with the BME280 sensor and a relay module for fan control.
  • Data Transport: MQTT protocol is used to publish sensor data and subscribe to control messages. The system utilizes topics for:
    • Publishing sensor data every 5 seconds.
    • Receiving control commands for the fan.
    • Setting temperature thresholds.
  • Visualization: A ThingWorx dashboard displays live data, including temperature, humidity, and the user-defined threshold. The dashboard also features time-series graphs for historical analysis.

3. Functional Workflow

  • The BME280 sensor continuously collects temperature, humidity, and pressure data, which is processed by the Raspberry Pi.
  • Data is published to an MQTT broker and visualized on the ThingWorx platform for user insights.
  • Farmers use the MQTT Dash app to:
    • Set temperature thresholds.
    • Turn the cooling fan on or off remotely.
  • When the temperature surpasses the set threshold, the fan is automatically activated to stabilize conditions. Notifications and status updates are provided on the dashboard and mobile app.

4. Technological Highlights

  • IoT Integration: Combines edge computing, MQTT, and cloud visualization for seamless monitoring and control.
  • Automation: Implements a feedback loop to autonomously adjust environmental conditions based on sensor readings.
  • Ease of Use: Provides farmers with simple controls via mobile apps and an intuitive dashboard.
